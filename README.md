# T-Shop Website

T-Shop is a responsive front-end e-commerce website built to practice modern web development concepts using HTML, CSS, and Bootstrap.

## Features
- Fully responsive design using Bootstrap 5
- Modern UI with gradient hero section
- Product cards with hover animations and shadow effects
- Multiple sections (Perfumes, Men's T-Shirts, Women's T-Shirts)
- Navigation bar with active links
- Contact page with form
- About page with company overview
- Social media integration (Facebook, Instagram, TikTok)

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome

## Purpose
This project was developed as a training project to improve front-end development skills, focusing on layout design, responsiveness, and UI/UX enhancements.
